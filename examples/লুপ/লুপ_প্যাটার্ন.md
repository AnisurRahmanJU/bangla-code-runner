### Without Newline
```py
ধরি ক;
ধরি খ;
ধরি গ = ইনপুট("ইংরেজিতে একটি নাম্বার নাও: ");

লুপ(ক = ১; ক <= গ; ক++)
{
   ধরি লাইন = " ";
    লুপ(খ = ১; খ <= ক; খ++)
    {
         লাইন =  লাইন + "  " + খ;
    }
    দেখাও(লাইন); 
}

```
### With Newline
```py
ধরি ক;
ধরি খ;
ধরি গ = ইনপুট("ইংরেজিতে একটি নাম্বার নাও: ");

লুপ(ক = ১; ক <= গ; ক++)
{
   ধরি লাইন = " ";
    লুপ(খ = ১; খ <= ক; খ++)
    {
        লাইন = লাইন + "      " + খ;
    }
    দেখাও(লাইন);
    // "\ন" হলো নিউলাইন
    দেখাও("\ন"); 
}
```
